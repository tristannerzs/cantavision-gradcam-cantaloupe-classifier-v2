Web-Deployed Cantaloupe Classification with ROI Cropping and Interactive Grad-CAM Explainability.

I made this complete academic web application to classify cantaloupe images through a browser-based machine learning system. The user can upload an image, select one exact fruit or leaf region, run prediction on that crop, and inspect the model focus using a Grad-CAM heatmap and overlay study panel.

Draw the orange box around the fruit. In ROI mode, the model classifies only that selected crop.

Academic model interpretation

Project Facts and Grad-CAM Meaning

Classification Task Binary image classification: cantaloupe versus not_cantaloupe.
Model Input The selected ROI is resized to 224 × 224 before prediction.
Confidence Score P(cantaloupe) represents how strongly the model predicts the selected crop is cantaloupe.
Threshold Rule If P(cantaloupe) is greater than or equal to the threshold, the final label becomes cantaloupe.
Grad-CAM Red / Yellow Red and yellow areas show the strongest Grad-CAM activations and the most influential visual regions.
Grad-CAM Blue Blue areas show weaker activation and usually contributed less to the final prediction.
Overlay View The overlay combines the selected crop with the heatmap to show where the model focused.
ROI Analysis ROI mode makes it possible to classify one fruit from an image containing many fruits.
